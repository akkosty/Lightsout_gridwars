# Lightsout Grid Wars

Игра "Светофильтр" в сетевом варианте.

## Описание
В этой игре игроки соревнуются в умении пропускать свет определенных цветов через фильтры. Задача - оставить на поле только фишки нужного цвета, остальные нужно "выключить" (превратить в черные).

## Правила
- На игровом поле расставлены цветные фишки.
- Выбор фишки переключает светофильтр, пропускающий только этот цвет.
- Фишка этого же цвета становится "мёртвой" (черной).
- Цель - оставить на поле только фишки одного заданного цвета.
